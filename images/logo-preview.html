Step 1 — Generate favicon files

1
Go to realfavicongenerator.net — upload logo-icon.svg. It generates every size you need in one zip file: favicon.ico, apple-touch-icon.png, android-chrome files, and a site.webmanifest.
2
Download the zip. Copy all files into the root of your website repo (same level as index.html — not inside a folder).
3
The generator gives you exact <link> tags to paste. Add them inside <head> in every HTML file on the site (index.html, checklist.html, thank-you.html, and any audit pages).

Step 2 — Replace logo in website header

1
Copy logo.svg and logo-reversed.svg into /images/ in your website repo.
2
In Claude Code, open index.html. Find the current logo — it's the text element in the header that reads "RemodelerRank". Replace it with: 
<a href="/" class="logo-link">
  <img src="/images/logo-reversed.svg"
       alt="RemodelerRank"
       height="36"
       width="auto">
</a>
Use logo-reversed.svg here because your header background is charcoal.
3
Do the same in checklist.html — find the .logo element and replace with the same img tag.
4
Push to GitHub. Netlify deploys automatically. Verify the logo appears on the live site and in the browser tab.
